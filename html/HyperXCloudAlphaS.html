HyperX Cloud Alpha S 電競耳機

  HyperX Cloud Alpha™ S 在 Cloud Alpha 突破性的基礎上再進化，帶來 HyperX 7.1 虛擬環繞音效1。透過隨附的音訊平衡控制器即可啟用環繞音效，並調整遊戲及通話音訊的平衡，無須進入任何選單。耳罩上新增低音調整控制，讓您可以自訂音效。Cloud Alpha S 沿用雙音腔驅動單體系統，能夠降低失真並提供更清晰的音質，同時也具備 HyperX 專屬舒適配戴體驗，記憶泡棉耳墊上的人造皮革也同步升級為更透氣的材質。

HyperX 7.1 環繞音效

  由 HyperX 調教並經由電競選手測試，確保玩家在遊戲中獲得更強化的音效，更豐富的聲音細節。NGENUITY 軟體自動偵測支援的遊戲，並選擇最佳化的環繞音效設定檔，提升電腦上的遊戲體驗。

透過低音調整控制設定個人化音效

  耳罩上的控制鈕，可選擇三種低音強度。完全開啟可放大低音效果，關閉可減少低音強度。

HyperX 雙音腔驅動單體

  將低音頻率從中音和高音區隔出來，再透過個別調校，使音域區別更加明顯，並可大幅減少失真。

遊戲和音訊平衡控制

  利用音訊平衡控制器輕鬆調整遊戲和通話音訊的平衡，無須經過繁瑣的系統選單設定。

HyperX 專屬舒適配戴體驗

  選擇最適合你的舒適度。升級後的人造皮革材質，具有 Cloud Alpha 耳墊的 8 倍透氣率，如果您希望有更好的透氣效果，包裝內附一組可替換的布質耳墊，讓長時間遊戲時更加舒適。

耐用鋁合金框架

  耐用的鋁製框架能夠耐受日常遊戲時的耗損。

進階音訊平衡控制器

  輕鬆自訂個人化音效。無須離開遊戲即可啟用 7.1 環繞音效、調整遊戲和通話音訊平衡、靜音麥克風和調整音量。

可拆式降噪麥克風和音源線

  這款耳機經由 Discord 和 TeamSpeak™ 認證，可輸出清晰的語音通話音質，並且配備可拆卸、抗磨損的線材。麥克風可自由彎曲，也可在聆聽音樂時卸除。
