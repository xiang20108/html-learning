HyperX Cloud Flight S 無線電競耳機

  HyperX Cloud Flight S 電池續航力高達 30 小時1，讓您盡情享受遊戲不間斷。這是首款 Qi 充電無線電競耳機，可為您提供無線充電和 2.4GHz 無線音訊傳輸的便利。按壓耳罩上的控制鍵即可啟用 HyperX 特調 7.1 虛擬環繞音效2，立即體驗更徹底的沉浸感。使用耳罩上的控制鍵輕鬆調整遊戲和通話音量，無須透過繁瑣的系統選單。也可以透過 HyperX NGENUITY 軟體，依照個人需求自訂左耳罩上四個控制鍵的個別功能。

  紓壓記憶泡棉和透氣人造皮革為 Cloud Flight S 提供了 HyperX 獨家舒適度。堅固的可調式鋼芯頭帶，在日常使用中更添耐用性。本產品經過 Discord 和 TeamSpeak 認證，並配備可拆式麥克風，且內建麥克風監聽 (Sidetone 側音) 功能。透過麥克風上的 LED 指示燈，快速辨別麥克風靜音/收音狀態。耳罩可旋轉 90°，輕鬆地掛置於頸部，創新的卡榫式設計能夠讓耳機在無線充電時保持穩定。

2.4GHz無線傳輸和30小時電池續航力

  快速傳輸帶來無線自由超長效電力讓遊戲時光不間斷，減少充電，爽快暢玩。

Qi 認證無線充電

  將耳罩旋轉至定位，即可透過左耳罩內建的 Qi 充電感應線圈進行充電。以無線方式為耳機充電，享受極致的無線便利。

HyperX 特調 7.1 虛擬環繞音效

  完全針對玩家的需求進行特調，HyperX 7.1 虛擬環繞音效讓您充分享受 PC、PS4 和 PS4 Pro 的遊戲體驗，徹底沉浸其中。

遊戲和語音音效平衡

  透過耳罩上的控制鍵快速調整遊戲和通話音量，讓您無須在繁瑣的系統選單中浪費時間。

HyperX 專屬舒適配戴體驗和耐用性

  紓壓記憶泡棉和透氣人造皮革，即使徹夜打 game 一樣保持舒適度。堅固的可調式鋼芯頭帶，在日常使用中更添耐用性。

可拆式麥克風配備 LED 靜音/收音指示燈

  經 TeamSpeak 和 Discord 認證，確保您與隊友通話的清晰音質，並內建麥克風監聽 (Sidetone 側音) 功能。透過麥克風上的 LED 指示燈，快速辨別麥克風靜音/收音狀態。當您結束遊戲並想要單純享受音樂時，可以直接將麥克風卸下。

可自訂左耳罩上的控制鍵功能

  使用易於操作的 HyperX NGENUITY 軟體，依照個人喜好設定內控制鍵功能。

90° 旋轉耳罩

  旋轉式耳罩讓您在休息時可將耳機舒適地掛置於頸部。
